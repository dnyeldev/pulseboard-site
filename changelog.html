The biggest update since launch. Smarter AI, more data, better everything.

◈
Pulse AI
New Structured fuel data cards. Prices render as visual tables with brand rows, change pills, and color-coded gasoline/diesel sections.
New Contextual quick action pills. Follow-up suggestions adapt to the topic: fuel, weather, earthquakes, markets.
New Typing indicator. Three-dot animation while AI processes your request.
New Chat bubbles. User messages right-aligned with red tint, AI responses left-aligned with PulseBoard avatar.
New Offline detection. Shows "No connection" screen when offline, auto-dismissing banner when connection returns mid-chat.
Fix Removed refreshSession() call on every AI message. Was eating 10s+ of the 15s timeout, causing "trouble connecting" errors.
Change Timeout increased from 15s to 30s. Claude API responses need room to breathe.
Change Errors now logged to Sentry instead of silently swallowed.
★
Monetization
New 7-day Pro trial. Opt-in with clear consent screen. No credit card required.
New Smart paywalls. 6th widget and Pulse AI gated for free users. CTAs: guests see "Sign up free", free users see "Upgrade to Pro".
New Rate limiting. Guest/Free: 2 chats/day. Trial: 30/day. Pro: 50/day. Per-minute cap prevents abuse.
New 500 character message cap. Silently truncated server-side. Character counter appears at 400+.
New Subscription offline caching. Trial/Pro users keep access during network drops. 24h TTL before fallback to free.
▦
Widgets
New Toll Fees widget. 160+ routes across 10 expressways. Full TRB data in Supabase.
New Daily Essentials widget. Rice, electricity, jeepney fare, eggs. 2x2 glass cards. Automated weekly via Edge Function.
New Live TV: News5 added as 7th channel. Pinned column prevents cron overwrites.
Change Fuel Prices: 3-tier data sourcing. DOE official data first, 5 RSS feeds backup, brand websites fallback. Never stale again.
Change Fuel Prices: responsive font sizing via scale(). Prices fit on one line across all screen sizes.
Change Market Radar: USD/PHP as hero card with full-width layout. 2x2 grid below with BTC, Gold, PSEi, Brent Oil.
Change Weather widget: new layout with large temp, high/low arrows, humidity droplet.
Fix Earthquakes: empty state when 0 quakes in 24h. No more showing 6-day-old data as if it's current.
Fix Live TV: green dot only pulses when stream is actually live. No more false "online" indicators.
Fix Alerts: PH-only filtering. No more Myanmar or Indonesia news showing up.

Change Hero section: 2-line greeting with name. Greeting pool capped at 2-3 words max. No more wrapping.

Security & Performance
Security XSS sanitization on Live TV videoId input.
Security Removed dangerous "Auth write access" ALL policy on live_tv_streams. Any logged-in user could previously overwrite stream data.
Security RLS verified on all 8 Supabase tables. user_profiles restricted to SELECT own data only.
New Sentry crash monitoring. JS errors, native crashes, and performance tracking. Source maps for readable stack traces.
New Better Stack uptime monitoring. 4 Edge Function endpoints checked every 3 minutes.
Perf fetchWithTimeout on all external API calls. No more hanging requests.
Perf Stripped 68 console.log statements across 15 files.
Perf Deleted 4 unused components. 396 lines of dead code removed.
Fix SSO nonce mismatch. Google: enabled skip nonce checks. Apple: proper raw/hashed nonce handling via expo-crypto.
Fix Pulse AI Edge Function 500. Env var name mismatch: code referenced ANTHROPIC_API_KEY, secret was CLAUDE_API_KEY.

Infrastructure
New ENV_VARS.md documenting all 17 environment variables across 10 categories.
New ErrorBoundary component. JS crashes show a recovery screen instead of a white screen of death.
Change RevenueCat: lazy require() for all 6 calls. Updated to react-native-purchases@9.15.1. No more Expo Go crashes.
Change Fuel price idempotency guard. Same-date runs are skipped automatically. No more double-apply data corruption.
Change Google Sign-In: isGoogleSignInAvailable now actually checks the native module instead of returning true.
Fix Fuel cache key fixed. Pulse AI can now read fuel data when answering price queries.
Fix Region key conflict resolved. Weather and fuel no longer overwrite each other's user region.
